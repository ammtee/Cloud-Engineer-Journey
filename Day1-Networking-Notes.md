# Day 1 - Networking Fundamentals

## What is a Network?

A network is a collection of interconnected devices that communicate and share resources.

## Types of Networks

### LAN
Local Area Network

### WAN
Wide Area Network

## Internet Basics

The Internet is a global network connecting millions of devices.

## IP Address

An IP Address uniquely identifies a device on a network.

Examples:

- IPv4: 192.168.1.1
- IPv6: 2001:db8::1

## DNS

Domain Name System converts domain names into IP addresses.

Example:

google.com → 142.250.xxx.xxx
